## For linux

### install nodejs
#### Ubuntu/Debian
```bash
sudo apt update -y && sudo apt upgrade -y
sudo apt install curl -y
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs -y
```

### Install Git
#### Ubuntu/Debian
``` bash
sudo apt update -y && sudo apt upgrade -y
sudo apt install git -y
```
